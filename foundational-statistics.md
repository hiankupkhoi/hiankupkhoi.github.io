# Foundational Statistics

Master the fundamentals of statistical analysis and research methodology. This course provides a comprehensive introduction to statistical concepts essential for data analysis and research.

## Course Topics

### 1. Data Collection and Descriptive Statistics
- Types of data and measurement scales
- Data collection methods
- Descriptive statistics: mean, median, mode
- Measures of dispersion and variation
- Data visualization techniques

### 2. Probability Theory
- Basic probability concepts
- Probability distributions
- Binomial, normal, and other distributions
- Central limit theorem
- Sampling distributions

### 3. Hypothesis Testing
- Formulating hypotheses
- Type I and Type II errors
- Significance levels
- One-sample and two-sample tests
- t-tests, chi-square tests, and ANOVA

### 4. Confidence Intervals and Estimation
- Point estimation
- Interval estimation
- Confidence intervals for means and proportions
- Standard errors

### 5. Introduction to Regression Analysis
- Linear regression basics
- Correlation and causation
- Simple linear regression
- Multiple regression
- Model diagnostics

### 6. Research Methodology
- Research design
- Sampling techniques
- Data quality and validation
- Ethical considerations in research
- Reproducible research practices

## Learning Outcomes

By the end of this course, you will be able to:
- Understand fundamental statistical concepts
- Apply appropriate statistical tests to research questions
- Interpret statistical results correctly
- Design and conduct basic statistical analyses
- Understand and avoid common statistical pitfalls

## Prerequisites

No prior statistical knowledge required. Basic mathematics skills recommended.
