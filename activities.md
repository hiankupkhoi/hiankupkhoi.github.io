# Activities

## Upcoming Workshops and Seminars

### Foundational Statistics Workshop
A comprehensive introduction to statistical concepts and methods for researchers and students.

**Topics:**
- Descriptive statistics and data exploration
- Probability distributions
- Hypothesis testing
- Confidence intervals and estimation

### Stata Training Series
Hands-on training sessions on data analysis using Stata.

**Modules:**
- Introduction to Stata
- Data management and manipulation
- Statistical analysis
- Publication-ready graphics
- Econometric modeling

### R Programming for Data Analysis
Learn R programming for statistical analysis and visualization.

**Modules:**
- R basics and data structures
- Data manipulation with dplyr
- Visualization with ggplot2
- Statistical analysis
- Advanced topics

### Special Seminars
Regular seminars featuring research presentations, methodological discussions, and expert talks on:
- Applied econometrics
- Causal inference
- Geospatial analysis
- Latest developments in data science
